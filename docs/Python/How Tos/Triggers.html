Triggers¶

The following is a list of available triggers. Any text in square brackets ([]) indicates it is optional, and any text in CAPITAL LETTERS should be replaced with a value specific to your use case.

"Item ITEM_NAME changed [from STATE] [to STATE]"¶

"Item ITEM_NAME received update [STATE]"
"Item ITEM_NAME received command [COMMAND]"
