Rules¶

core.rules.rule(name=None, description=None, tags=None)¶

openHAB DSL style rule decorator.

See How Tos/Rules:Decorators for a full description of how to use this decorator.

Examples

@rule('name', 'description', ['tag1', 'tag2'])
@rule('name', tags=['tag1', 'tag2'])
@rule('name')
Parameters

  • name (str) – Display name of the rule.

  • description (str) – Description of the rule.

  • tags (list[str]) – List of tags.

core.rules.addRule(rule)¶

Adds rule to openHAB’s ruleRegistry.

This is a wrapper of automationManager.addRule() that does not require any additional imports. See How Tos/Rules:Extensions for examples of how to use this function.

Parameters

rule (Rule) – A rule to add to openHAB.
